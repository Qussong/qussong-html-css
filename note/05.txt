<!-- 이미지, 오디오, 비디오, 하이퍼링크 삽입 -->

- <img>: 웹 문서에 이미지 삽입시 사용하는 tag
  <img src="이미지 주소 값" alt="대체 텍스트">
  이미지 주소 값의 경우 이미지의 위치에 따라 주소 작성법이 달라진다
  이미지는 시각자료이기에 시각장애인의 경우 이미지를 볼 수 없다.
  때문에 해당 이미지를 대체해서 읽어 줄 수 있는 대체 텍스트를 넣어줘야 한다.
  이미지의 주소가 잘못 들어간 경우엔 이미지대신 대체 텍스트가 화면에 나타난다.
  아이콘이나 특별한 의미가 있는 이미지가 아닌경우 대체 텍스트는 넣어주는 것이 좋다

- <audio>: 오디오 파일을 삽입해주는 tag
  <audio src="오디오 주소 값" controls></audio>
  이미지와 거의 동일하며, 이미지와 달리 대체 텍스트가 없다
  controls 속성을 입력해주면 재생막대가 생성된다.

- <video>: 비디오 파일을 삽입해주는 tag
  <video src="비디오 주소 값" controls width="크기 값"></vidoe>
  audio tag와 동일하고 영상의 크기를 width 속성으로 조절 할 수 있다.

- <audio>, <video> tag의 속성
  controls: 플레이어 화면에 컨트롤 바를 표시
  autoplay: 오디오나 비디오를 자동으로 실행
    비디오의 경우 chrome 브라우저와 휴대폰의 경우 자동재생이 되지 않는다.
    만약 사용하고자 한다면 muted 속성을 꼭 같이 넣어줘야 한다.
    오디오의 경우 muted 속성이 없어도 자동 재생된다.
  loop: 오디오나 비디오를 반복 재생한다
  muted: 오디오나 비디오의 소리를 제거
  preload: 페이지를 불러올 때 오디오나 비디오 파일을 어떻게 로딩할 것이지 지정한다.
    사용할 수 있는 값은 auto, metadata, none으로 기본값은 preload="auto"
  width, height: 비디오 플레이어의 너비와 높이를 지정한다.
    width와 height의 값 중에서 하나만 지정하면 나머지는 자동으로 계산된다.
  poster="파일이름": <video> tag에서 사용하는 속성으로, 
    비디오가 재생되기 전까지 화면에 표시될 포스터 이미지를 지정한다.
    poster img 를 지정하지 않으면 비디오를 재생하기 전까지 비디오 화면은 검은색 창이다.
    
- <a>: 하이퍼링크를 삽입해 주기위해 사용하는 tag
  <a href="주소 값"> 텍스트 내용</a>
  주소 값 뒤에 target="_blank" 속성을 붙여주면 새탭에 링크된 주소가 열린다
