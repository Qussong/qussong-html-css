< 변형(transform) > 
  : 특정 요소의 크기나 형태 등 스타일이 바뀌는 것

- 2차원적인 변형
    수평이나, 수직으로 웹 요소 변형
    크기나 각도만 지정하면 됨
    2차원 좌표 사용

- 3차원적인 변형
    x,y축에 원근감 추가
    z축은 앞뒤로 이동, 앞으로 올수록 값이 더 커짐

- 선택자 {transform: 속성 값;}
  -> 속성 값 종류
    - translate(x, y) : 지정한 크기만큼 x축, y축으로 이동한다
    - translateX(x) : 지정한 크기만큼 x축으로 이동
    - translateY(y) : 지정한 크기만큼 y축으로 이동
    - 단위 : px

    - scale(x, y) : 지정한 크기만큼 x축과 y축으로 확대, 축소한다
    - scaleX(x) : 지정한 크기만큼 x축으로 확대, 축소한다
    - scaleY(y) : 지정한 크기만큼 x축으로 확대, 축소한다
    - 단위 없음

    - rotate(각도) : 
    - 지정한 각도만큼 회전(+:시계방항, -:시계반대방향)
    - 단위 : deg
      -> 입체감 있게 회전 하도록 설정할 수 있다.
          perspective : 
            - 원근감 추가 0보다 
            - 큰 숫자를 적어줘야한다
            - rotate 하고자 하는 객체의 부모요소에 해당 속성을 넣어줘야 한다.
            - 회전하는 객체에 perspective를 적용하면 안된다

    - skew(x, y) : 지정한 각도만큼 x축과 y축으로 왜곡한다
        -> 보통 포토샵을 활용하지 않고 글자를 비틀때 사용한다.
    - skewX(x) : 지정한 각도만큼 x축으로 왜곡한다
    - skewY(y) : 지정한 각도만큼 y축으로 왜곡한다
    - 단위 : deg
