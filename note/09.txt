< 글꼴과 관련된 속성 >
- font-family
- font-size
  em : 영어 알파벳 대문자 M의 폭 만큼의 크기를 1em이라고 한다. 1em = 16px
- font-style : 
  이텔릭체로 나타내고자 할때 사용하는 속성
- font-weight : 
  폰트의 굵기를 설정하는 속성, 기본 값은 400, 400보다 커지면 굵어진다

< 웹폰트 >
- 웹폰트를 사용하면 다양한 폰트를 사용할 수 있다.
- 사용자 시스템에 설치되어 있지 않은 폰트를 의미한다.
- link를 해서 사용하게 되는데 이 경우 네트워크의 연결에 많은 영향을 받는다
- 많이 사용할 경우 웹 사이트의 속도에 영향을 줄 수 있다.

● 웹 폰트 사용하는 방법 (google fonts)
  - google fonts 접속 
  - 마음에 드는 폰트 찾아 클릭 
  - select this style 클릭
  - link tag와 import tag 중 원하는 tag를 선택
  - 아래 나오는 style tag 안의 내용을 복사
  - 작성중인 파일의 style tag에 붙여넣기(style tag 바로 아래 추천) 
  - CSS rules to specify famlies 를 복사해서 원하는 선택자에 붙여넣기한다.
  - 해당 선택자가 변경된다
  - 만약 네트워크에 문제가 있어 웹 폰트가 사용할 수 없는 경우에 대비해 뒤에 시스템에 설치되어 있는 다른 글자체를 추가해준다.

< color 속성 >
- web 에서 색상을 표현하는 방법은 다양한다
1. 색상 이름 표기법
  - 잘 알려진 색상 이름으로 표시, 기본 16가지
  - 웹 안전 색상(web-safe-color)
    : 모든 브라우저에서 표현할 수 있는 색상
      기본 16가지 색상을 포함해 모두 216가지 

2. 16진수 표기법
  - #과 함께 6자리의 16진수로 표시
  - 앞에서부터 두자리씩 묶어 r,g,b의 정도를 뜻함
  - 000000(black) ~ ffffff(white)
  - 두자리씩 중복될 경우 줄여 사용할 수 있다.
    ex) #cccccc = #ccc
  
3. hsl/hsla 표기법
  - color.hsl(240, 100%, 50%) 처름 3자리 숫자로 표시
  - 순서대로 색상(hue), 채도(saturation), 밝기(light)를 뜻함
  - 투명도를 조절할 때는 마지막에 알파값 추가
    ex) hsla(240, 100%, 50%, 0.3)
  - 알파값은 불투명도를 나타내는 값으로 0~1 값 중에서 사용
    (1은 불투명, 0은 오나전 투명)

4. rgb/rgba 표기법
  - color.rgb(255,0,0)처럼 세자리의 숫자로 표시
  - 순서대로 red, green, blue를 의미함
  - 하나도 섞이지 않았을 때는 0, 가득 섞였을 때는 255
  - 투명도를 조절할 때는 마지막에 알파값 추가해준다
    ex) color.rgb(255,0,0,0.3)

< tip >
- colorbox에 마우스를 올리면 색상을 선택할 수 있는 표가 나온다
- 해당 표의 상단 부분 rgb 값이 나와 있는 란을 누르면 
  16진수 표기법 -> hsl 표기법 -> rgb 표기법 이 순차적으로 나온다
